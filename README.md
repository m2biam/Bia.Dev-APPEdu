# 🐾 Jogos Infantis PWA - Plataforma Educativa

Um Progressive Web App (PWA) de jogos educativos infantis, desenvolvido com foco em performance, responsividade e usabilidade para crianças. O aplicativo foi projetado para funcionar offline, ser instalável em dispositivos móveis e oferecer uma experiência segura de aprendizagem.

## 🚀 Funcionalidades

- **Jogo da Memória:** Cartas ilustradas com animais, lógica de embaralhamento e validação de pares.
- **Caça-Palavras (Multinível):** Níveis Fácil, Médio e Difícil, com sistema de áudio para acertos e proteção de cliques.
- **PWA (Progressive Web App):** Manifesto e Service Worker configurados para instalação direta no smartphone e funcionamento Offline.
- **Design Responsivo:** Interface Mobile First que se adapta a diferentes tamanhos de ecrã.
- **Feedback Visual e Sonoro:** Modais de vitória e efeitos sonoros para incentivar a criança.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica.
- **CSS3:** Flexbox, Grid e Media Queries para responsividade.
- **JavaScript (ES6+):** Lógica do jogo, manipulação do DOM e gestão de estado.
- **Service Workers:** Para suporte offline e cache de recursos.
